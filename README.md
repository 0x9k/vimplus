安装配置
-------------

> git clone https://github.com/chxuan/vimplus.git
> ./vimplus/setup.sh

运行setup.sh脚本程序将会自动安装并配置好vim，安装大约需要花费40分钟，主要是下载编译[Valloric/YouCompleteMe](https://github.com/Valloric/YouCompleteMe)比较耗时，请耐心等待直到安装完成^_^
